== Description ==

The Custom Styles for Guten-berg Block enhance your WordPress blocks and apply Custom CSS styles to your wordpress Gutenberg Blocks.
We initially designed some of the components which are listed below.

<ul>
<li>Guten-berg Columns styles</li>
<li>Guten-berg Spacer styles</li>
<li>Magic Block (creates div with desired classes) styles</li>
</ul>

== Installation ==
<ul>
<li>Upload the entire custom-styles-guten-berg-block folder to the /wp-content/plugins/ directory.</li>
<li>Activate the plugin through the ‘Plugins’ menu within the WordPress Dashboard.</li>
</ul>
You will now see the Gutenberg Block Custom Styles working in your WordPress pages and posts.
